# mpad

A native Markdown app for macOS.

Write in a clean WYSIWYG editor, keep plain `.md` files on disk, and review Git changes without leaving the app.

![MIT License](https://img.shields.io/badge/license-MIT-blue)

## Features

- **Write faster** - Rich editing without memorizing Markdown syntax
- **Own your files** - Regular Markdown files, no lock-in
- **Stay in flow** - Command palette, find, and keyboard-first workflow
- **See Git context** - Status, diff, and history while you write
- **Open from terminal** - Launch files with `mpad your-file.md`

## Install (macOS)

```bash
curl -fsSL https://entire.io/install.sh | bash
```

This installs:

- `/Applications/mpad.app`
- `/usr/local/bin/mpad` (CLI launcher)

Or download the latest build from [GitHub Releases](https://github.com/eneakllomollari/mpad/releases).

## Quick Start

```bash
mpad README.md
```

Or open the app from Applications and choose a file/folder.
